Animation

There's a large selection of premade animations to use with the framework as well as a easy to use system for applying animations to elements in your project

Animation classes are added under the anim attribute. This method helps make it clear which elements have animations and which don't. The first part in the anim attribute should be the name

Animation Types

  • blink
  • fade_in
  • shake
  • slide_in_back
  • slide_in_top
  • slide_in_bottom
