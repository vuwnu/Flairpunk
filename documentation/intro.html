Framework Naming Conventions

To keep this framework consistent, class names and variable names will all follow a set naming structure. This can be seen below, along with an explanation on how different characters are used

[property]-[value]

Underscores

Underscores are used exclusively for adding spaces between words such as dark_blue or space_evenly. Dashes are used for linking parts of a name together, such as the property and the value you are setting
