Layout System

With the layout system you can easily give your site an appealing design

To start using the layout system simply add a use-layout class to your body element along with the name of layout. All available layouts can be seen below

Header

The header element is added as part of the layout. It will always appear at the very top of the page, no matter where it is placed in the html

Main

The main element signifies the main content of the page. This should be unique to each page

Footer

The footer element is also usually added as part of a layout. It will be placed at the bottom of your page
