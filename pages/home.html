Features

CSS Variables

Gluey makes use of CSS Variables to make altering the theme of your site very straightforward

Layout System

With the layout system you can create great looking websites incredibly easily. Making use of CSS Grid, there are a wide variety of different layouts to choose from.

Helper Classes

There are a wide range of helper classes available that help with making for precise adjustments to the look of your site. These classes can be used to overwrite the default values of component classes

Component Classes

With component classes you can quickly build your website using components

The Pillars

Responsiveness

A site built with zoon should work across all devices and all browsers. Everything should have fallbacks available in case fonts or certain stylings are unavailable.

Contextuality

Not every part of a style should need to be applied manually. zoon will apply certain stylings for you automatically based on the context of what you're writing.

Presentation

Not only do I want zoon to function well, I want it to look good too. Using zoon it should be easy to build a site that works as well as it looks.
